# godot_projects
This repository contains a growing collection of small games and prototypes built with the Godot Engine.
Each project focuses on learning a specific mechanic, UI concept, or gameplay system.
Instead of building one large game, I use this repository to explore different ideas, experiment with Godot’s features, and document my progress as I learn.

All projects are self‑contained inside their own folders and can be opened directly in Godot.
